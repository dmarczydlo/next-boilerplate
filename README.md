# next-boilerplate

[![Greenkeeper badge](https://badges.greenkeeper.io/dmarczydlo/next-boilerplate.svg)](https://greenkeeper.io/)

Include: 
* SaSS
* Redux
* Nextjs
* Router
* Nodejs (custom server)

## Getting started
1. ```git clone git@github.com:dmarczydlo/next-boilerplate.git```
1. ```yarn install```
1. ```yarn dev```
