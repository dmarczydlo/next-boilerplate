# next-boilerplate
Include: 
* SaSS
* Redux
* Nextjs
* Router
* Nodejs (custom server)

## Getting started
1. ```git clone git@github.com:dmarczydlo/next-boilerplate.git```
1. ```yarn install```
1. ```yarn dev```
